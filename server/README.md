# WhatsApp Web Backend

## Setup & Run

```bash
cd server
npm install
npm start
```

The server will start on **http://localhost:3001**

## Requirements
- Node.js 18+
- Google Chrome or Chromium installed (used by Puppeteer)

## First Run
1. Start the backend: `cd server && npm install && npm start`
2. Start the frontend: `npm run dev` (from project root)
3. Open http://localhost:5173
4. Scan the QR code with your WhatsApp mobile app
5. Messages and chats will load automatically
